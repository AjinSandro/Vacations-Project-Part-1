1 - Install env through:  python -m venv env   
2 - Install dependencies through requirements.txt: pip install -r requirements.txt 
3 - Activate env while being outside of the folder in the terminal: env/scripts/activate            
4 - Run the program: flask --app app.py run --debug --port 5000
5 - Enter the given IP for the site through the terminal and enjoy viewing nice beaches and cool ocean waves.
